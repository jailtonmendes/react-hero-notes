##  Como baixar o projeto

```bash
    # Clonar o repositório
    $ git clone https://github.com/jailtonmendes/react-hero-notes

    # Entrar no diretório
    $ cd react-hero-notes

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ yarn dev
```
